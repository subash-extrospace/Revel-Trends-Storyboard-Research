# Abstract

This research presents REVEL-TRENDS, a comprehensive generative AI ecosystem designed to automate and enhance the creation of promotional video content for businesses. The system integrates user-driven context analysis, retrieval-augmented generation, and a multi-agent large language model (LLM) framework to produce creative, practical, and high-quality marketing concepts. The process begins with a structured context analysis module, where a conversational agent gathers detailed business information, including target audience, goals, challenges, and production constraints. This contextual data is enriched through a vector similarity search over a curated database of high-performing promotional content, enabling data-informed ideation. The core contribution lies in the LLM Council architecture, a multi-stage ideation pipeline consisting of four specialized agents: an Ideator that generates diverse concepts, a Critic that refines and improves them, a Moderator that aligns ideas with evaluation metrics, and Judges that select the most promising outputs. This collaborative framework ensures both creativity and quality control, resulting in 4–8 optimized video ideas. The system further incorporates human-in-the-loop refinement, allowing users to select, merge, and personalize ideas. These refined concepts are transformed into detailed, scene-by-scene scripts through a story generation module, augmented by iterative brainstorming suggestions that expand narrative possibilities. Finally, the system supports optional AI-driven video or image generation, converting scripts into visual outputs via prompt engineering. REVEL-TRENDS demonstrates how generative AI, combined with structured reasoning, retrieval mechanisms, and human feedback, can streamline creative workflows and empower businesses to produce engaging, platform-optimized promotional content with minimal resources.
